## Usage

### HTML
```html
<img src="https://cdn.jsdelivr.net/gh/micharl13/icons@main/icons/dark/24px/0-circle-outline.svg" alt="0 outline icon">
````

### Inline SVG

```html
<svg>
  <!-- paste SVG code here -->
</svg>
```
